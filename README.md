# 🔐 Cipher Panel Web Application

A modern encryption and decryption web application designed to secure and transform text using real cryptographic techniques. Built with a focus on cybersecurity concepts, interactive UI, and real-time processing.

---

## 🚀 Features

✅ Real Encryption & Decryption

* Supports secure text transformation
* Implements actual cipher logic (not fake UI-based encryption)

✅ Multiple Cipher Techniques

* Caesar Cipher
* Custom text transformation logic
* Expandable for advanced algorithms

✅ Interactive UI

* Real-time input/output updates
* Matrix-style background animation
* Clean and modern interface

✅ Error Handling

* Prevents empty input encryption
* Handles invalid data gracefully

---

## 🛠️ Tech Stack

* Python (Flask)

## ⚡ How to Run

1. Clone the repository
   git clone https://github.com/sunilsolanky230-ux/CIPHER-PANEL-WEB-APPLICATION

2. Navigate to project folder
   cd cipher-panel

3. Install dependencies
   pip install flask

4. Run the application
   python app.py

5. Open in browser
   http://127.0.0.1:5000/
